<h1>osTicket - Post-Install Configuration</h1>
This is an outline of the post-install configuration of the open-source help desk ticketing system osTicket.<br />




<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Internet Information Services (IIS)

<h2>Operating Systems Used </h2>

- Windows 10</b> (21H2)

<h2>Post-Install Configuration Objectives</h2>

- Configure Roles
- Configure Departments
- Configure Teams
- Comfigure Agents
- Configure Service level Agreemens(SLA)

<h2>Configuration Steps</h2>

<p>
  Configure Role

  This is grouping permissions together and giving permissions to certain people. Admin Panel->Agents->roles
